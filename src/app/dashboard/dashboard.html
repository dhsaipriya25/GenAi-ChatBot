Welcome to Gen AI Chatbot


A chatbot is essentially a computer program designed to simulate human conversation. It can interact with users through text providing information, completing tasks, or engaging in casual conversation.

Get Started


Steps for Upload a documents

Step 1:Start a new Chatbot---Click the "UPLOAD File" button to start the process.

Step 2:Upload your Documents---Here you can upload the documents you want your chatbot to use. It supports formats like PDF,DOC,TXT. Select and upload your documents here. And provide specific API Key, Application name, Prompt and soo on like required fields.

Step 3:Acknowledge page---After clicking on submit button in upload documents page then you will redirect to Acknowledge page, in that you got all the details which you entered in upload documents page and it will generate one Unique ID along that details.Based on that Unique ID your Chatbot starts.

Step 4:Build the chatbot--- Once your documents are uploaded, click "Submit" to build your chatbot. This involves processing your documents to create a searchable database for your uploaded files effectively.

Step 5:Interact with Your Chatbot---When the chatbot is ready. you'll be taken to a chat interface where you can ask questions about your uploaded documents.

Key Points


API KEY--- unique codes that are passed to APIs to identify the applications, users, or developers that are making calls.

URL--- A URL (Uniform Resource Locator) is a unique identifier used to locate a resource on the Internet.

FILE--The input file can be PDF, DOC,TXT etc.. based on that Chatbot will the analyze your file and answer your question

APPLICATION NAME--- Application Name must be the file name

PROMPTS---By default we have to some prompts related to your file name and application name.
